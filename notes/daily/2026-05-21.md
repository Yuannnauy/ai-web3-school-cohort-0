## Core Takeaways

- [ ] **RAG is an evidence chain, not just a vector DB.** Chunk → Vectorize → Retrieve → Rerank → Cite. If any link breaks, hallucination just moved from inside the model to inside your retrieval system.
- [ ] **Agent = constrained execution loop, not autonomy.** A real Agent needs all five: goal, tools, state, permissions, stop conditions. Model + tools alone = a talking script executor.
- [ ] **RAG feeds Agent's eyes; Agent is RAG's hands.** RAG fetches current, cited information before Agent acts. Agent acts on that information within defined guardrails.
