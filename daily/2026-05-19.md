## Core Takeaways

- [ ] **LLM is a reasoning layer, not a truth source.** Model outputs are probabilistic candidates, not facts. Real facts must come from databases, APIs, logs, or other trusted systems.
- [ ] **Prompt is an interface protocol, not a magic spell.** A good prompt tells the model when to say "I don't know" instead of confidently hallucinating.
- [ ] **Blockchain is a state machine.** The network doesn't "store data" — it lets trustless participants agree on state transitions, block by block.
